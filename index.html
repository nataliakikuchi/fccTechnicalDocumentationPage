Customer Web App

Obs.: Este projeto era chamado "Front" até o dia 10/01/2019. Algumas configurações do CircleCI, da AWS e outros ainda podem se referir ao nome antigo.

Introduzindo

O projeto customer-web-app é um SPA construído em Ember para ser a interface web que o cliente final da Creditas acessa em todo o processo de originação (ou seja, da criação do lead até ser pago o empréstimo). O deploy é feito no sub-domínio app.creditas.com.br (ambiente de staging em app-staging.creditas.com.br).

Requisitos

Para rodar o projeto você precisa ter instalado em seu computador:

  • Git
  • Node.js (com NPM)
  • Bower
  • Ember CLI
  • PhantomJS
Instalar

-> Clonar esse repositório localmente:

(with SSH) git clone git@github.com:Creditas/customer-web-app.git
(with HTTPS) git clone https://github.com/Creditas/customer-web-app.git

-> Mudar para o diretório criado cd customer-web-app
-> npm install
-> bower install

Rodar o projeto para desenvolvimento

-> Usando os hosts em ambiente staging do core e do auth

AUTH_HOST=https://auth-staging.creditas.com.br ember server --proxy=https://core-staging.creditas.com.br

Você deve apontar a variável de ambiente AUTH_HOST para o endereço staging do servidor de autenticação. Sem esse valor, o servidor de autenticação usado pela flag --proxy pode gerar comportamentos inesperados.

-> Visite a aplicação em: http://localhost:4200

Usando o mirage como servidor (NÃO USAR: o mirage não está propriamente configurado neste projeto atualmente)

-> ember server

-> Visite a aplicação em: http://localhost:4200

Rodar os testes

-> ember test ou
-> ember test --server ou
-> Com o projeto rodando, acessar http://localhost:4200/tests

Testes End-to-End

Nós também rodamos um teste end-to-end(e2e) do fluxo de cadastro no projeto: https://github.com/Creditas/tests-e2e-front

O teste e2e roda no CircleCI a cada deploy no ambiente staging.

É importante acompanhar se o teste e2e está passando sempre e identificar possíveis erros. Em geral, chega uma notificação no email quando o teste quebra, mas é possível checar diretamente no site do CircleCI.
